# CSS Gerador com IA ⚡

Um gerador de código CSS e HTML inovador que utiliza inteligência artificial para transformar suas ideias em código funcional.

## 🎯 Sobre o Projeto

Descreva o que você imagina e veja a mágica acontecer! Este projeto permite que você gere código HTML e CSS utilizando a API Groq com o modelo Llama 3.3 70B.

## 🚀 Funcionalidades

- ✨ Geração de código HTML e CSS por IA
- 🎨 Preview em tempo real
- 🔥 Interface moderna e responsiva
- 🎯 Prompts inteligentes para animações (quicando, girando, etc.)

## 📦 Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- Chave de API do Groq (obtenha em https://console.groq.com)

## 🔧 Instalação

1. Clone o repositório:
```bash
git clone https://github.com/PosterDev/Gerador-com-IA.git
cd Gerador-com-IA
```

2. Abra o arquivo `index.html` no seu navegador

3. Adicione sua chave de API Groq no arquivo `scripts.js`:
```javascript
let chave = "sua_chave_aqui";
```

## 📝 Como Usar

1. Digite uma descrição do que você deseja (ex: "Bola azul pingando")
2. Clique no botão "Gerar Código ⚡"
3. Veja o código gerado na seção "Código"
4. Visualize o resultado em tempo real no "Preview"

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura
- **CSS3** - Estilos
- **JavaScript** - Lógica
- **API Groq** - IA para geração de código
- **Modelo Llama 3.3 70B** - LLM

## 📂 Estrutura do Projeto

```
├── index.html       # Arquivo principal
├── styles.css       # Estilos da aplicação
├── scripts.js       # Lógica JavaScript
└── README.md        # Este arquivo
```

## 🔑 Configuração da API Groq

1. Acesse https://console.groq.com
2. Crie uma conta ou faça login
3. Gere uma chave de API
4. Adicione a chave no arquivo `scripts.js`

**IMPORTANTE:** Nunca compartilhe sua chave de API em repositórios públicos. Use variáveis de ambiente ou armazene localmente.
